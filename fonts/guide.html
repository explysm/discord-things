How to Create Your Own Fonts

How to Create Your Own Font Pack

This guide shows you how to create custom font packs for Discord mobile mod clients using pre-made fonts.

Step 1: Get Your Raw Font Links

Option A: Direct Download Link

If you got the font from a website, copy the direct download link.

Option B: Upload to File Hosting

Download the font file and upload it to one of these services:

  • Catbox.moe: Free file hosting for TTF files
  • GitHub: Upload to a GitHub repository and use raw link
  • Google Drive: Set file to public and get direct link
  • Dropbox: Generate public share link

Note: Make sure the links end in .ttf and are directly accessible.

Step 2: Create Your JSON File

Create a file named "font-pack-name.json"

Note that you can name it whatever you want, it just has to end with ".json"


Replace all placeholder values (text inside "") with your font information:

{ "spec": 1, "name": "Font Pack Name"", "previewText": "Preview text", "main": { "ggsans-Normal": "Normal Font Link", "ggsans-Bold": "Bold Font Link", "ggsans-Medium": "medium font link", "ggsans-Semibold": "semibold font link", "ggsans-ExtraBold": "extrabold font link", "ggsans-NormalItalic": "normal italic font link", "ggsans-BoldItalic": "bold italic font link", "ggsans-MediumItalic": "medium italic font link", "ggsans-SemiboldItalic": "semibold italic font link", "ggsans-ExtraBoldItalic": "extrabold italic font link", "NotoSans-Normal": "noto normal link", "NotoSans-Bold": "noto bold link"", "NotoSans-Medium": "noto medium link", "NotoSans-Semibold": "noto semibold link", "NotoSans-ExtraBold": "noto extrabold link", "ABCGintoNord-ExtraBold": "abcginto link", "SourceCodePro-Semibold": "sourcecodepro link" } }

What If I Don't Have All Font Variants?

No problem! If you don't have a specific font variant, you can:

  • Reuse existing variants: Use the same link for similar weights
  • Use closest available: For "Medium" you might use "Normal"
  • Remove unused keys: Simply delete lines you don't need
  • Use different font links: You can use different fonts for missing variants, or create a whole new font pack that includes the missing variants.

Step 3: Upload Your JSON File

Upload to a File Hosting Service

Upload your completed JSON file to any of these services:

  • Catbox.moe: Recommended - free and reliable
  • GitHub: Upload as a file in a repository
  • PasteBin: Use "Paste" and set to unlisted
  • Any public file host: As long as it provides a direct link

Important: Make sure you get the direct raw link to your JSON file.

Step 4: Import Font in Discord Mobile

Import Steps for Any Mod Client:

  1. Open your Discord mobile mod client
  2. Go to Font Settings
  3. Press the plus (+) button
  4. Select Import from link
  5. Paste your JSON file link
  6. Press Import
  7. Scroll down and press Import again to confirm

Font Design (Optional)

If you want to create your own fonts instead of using pre-made ones:

Font Creation Tools

  • FontForge: Free, open-source font editor
  • Glyphs: Professional font editor (Mac)
  • RoboFont: Professional font creation tool
  • Adobe Illustrator: With font export plugins

Export Requirements

  • Export in TTF format
  • Include basic Latin alphabet (A-Z, a-z)
  • Include numbers (0-9) and punctuation
  • Test before using

Common Issues

"Invalid JSON" Error

Cause: JSON syntax error or missing quotes/commas

Solution: Use a JSON validator like jsonlint.com to check your file

"Font Not Loading"

Cause: Broken font link or wrong file format

Solution: Test all font links in browser first

"Import Failed"

Cause: JSON link not accessible or expired

Solution: Use a reliable file host like Catbox.moe

Font Looks Wrong

Cause: Wrong font variant mapped to wrong key

Solution: Double-check your JSON mappings

Pro Tips

Test Links First

Always test your font and JSON links in a browser before importing.

Backup Your JSON

Keep a copy of your JSON file locally for easy editing.

Preview Text

Use meaningful preview text that shows off your font's style.
